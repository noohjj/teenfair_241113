# 사이트 개요

1. 사이트 이름 : TEENFAIR

2. 사이트 개요 : 전국 청소년들이 자신의 거주 지역 주변의 상담복지센터를 쉽게 찾을 수 있도록 관련 사회복지 정보 제공

3. STACK : react, js, html, css

4. 기능 : 거주 지역을 검색하여 주변에 위도 경도에 맞추어 주변의 청소년상담복지센터를 쉽게 찾을 수 있도록 함, 또한 해당 청소년상담복지센터의 정보 제공
